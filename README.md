# Last FM Page

For DEMO click [HERE](https://antonskliarov.github.io/last-fm/#/)

*****

## Tech stack

  * HTML5
  * Bulma, SCSS
  * JavaScript
  * React (v.17.0.1), React Router
  * Redux, Redux thunk

### Installing

1. Fork and clone this repository
2. Run `npm install` in your terminal
3. Run `npm start`
4. Open `http://localhost:3000/` at your browser
